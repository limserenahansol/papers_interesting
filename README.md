# papers_interesting

Archive of **paper discussions**: summaries, notes, and (when provided) source files or exports of the conversation.

## Layout

| Path | Purpose |
| --- | --- |
| [`entries/`](entries/) | One Markdown file per paper thread (summary + conversation excerpt + metadata) |
| [`files/`](files/) | Optional attachments (PDFs, excerpts) linked from entries; use sensible names |

## Entry file naming

`YYYY-MM-DD_short-topic.md` (example: `2026-03-25_fentanyl_vta_ensembles.md`)

Each entry should include:

- **User request** (or first message of the thread)
- **Paper** (title, authors, year, DOI/URL if known)
- **Summary** (your synthesis)
- **Conversation** (short transcript or bullet recap of what was discussed)
- **Files** (paths under `files/` if any were saved)

Entries are committed and pushed to this repo so you can search history on GitHub.
